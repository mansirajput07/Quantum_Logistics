## ğŸ”¬ Quantum Algorithms Used

### QAOA (Quantum Approximate Optimization Algorithm)
- Encodes optimization problems as quantum Hamiltonians
- Uses hybrid quantum-classical optimization
- Explores multiple solutions in parallel

### Quantum Machine Learning Models
- **VQC**: Variational Quantum Classifier for route classification
- **QSVM**: Quantum Support Vector Machine for pattern prediction
- **QGAN**: Quantum Generative Adversarial Network for traffic simulation
